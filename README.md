# Ahimsa Kitchen

MVP кастомного планировщика веганского питания — соберите день или неделю вручную, отслеживайте нутриенты.

## Возможности

- **3 языка:** English, Русский, Українська
- **Кастомный планировщик:** день или неделя, несколько блюд на приём пищи
- **Поиск рецептов** с фильтрами (кухня, регион, белок, теги: без глютена, много белка, железо…)
- **Drag & drop** блюд между приёмами и днями, копирование блюд и дней
- **Подобрать меню** — автозаполнение планировщика (рандомный генератор)
- **Экспорт** в текстовый файл и печать / PDF
- **38 рецептов** мировых кухонь
- **КБЖУ, клетчатка, железо, йод, D3** по секции, блюду и за день
- **Целевой нутриент** — отображение по блюдам и итогу
- **Образовательные врезки** о нутриентах
- **Замена блюд** на альтернативы из той же категории с пересчётом
- **Дисклеймеры** для опций железо / йод / D3
- Всё состояние хранится в `localStorage`, без бэкенда и регистрации

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## Стек

- React 19 + TypeScript
- Vite
- Клиентское хранилище (localStorage)

## Данные

Датасет ~50 веганских продуктов в `src/data/products.ts`. Каждая позиция содержит ссылку на источник (USDA FDC / EFSA) для будущей верификации.

## Границы MVP

Приложение **не** собирает медицинские данные, **не** предлагает похудение, **не** даёт медицинских советов и **не** рекомендует дозировки добавок.
